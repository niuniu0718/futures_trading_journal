# 期货交易记录系统 Windows版

## 版本历史
- v1.0.0 (2025-01-30): 初始版本
- v1.1.0 (2025-02-04): 新增时间筛选、品种管理、结算价对比等功能

## 系统要求
- Windows 7/8/10/11
- Python 3.8 或更高版本

## 安装步骤

### 1. 安装Python
如果还没有安装Python，请从以下地址下载安装：
https://www.python.org/downloads/

安装时请勾选 "Add Python to PATH"

### 2. 解压文件
将压缩包解压到任意目录，例如：`C:\futures_trading_journal\`

### 3. 启动系统
双击运行 `启动系统.bat` 即可

首次运行会自动安装依赖，请耐心等待。

## 访问地址
启动成功后，在浏览器中打开：
- http://localhost:5000

## 主要功能

### 1. 交易记录管理
- 新建交易记录
- 编辑交易记录
- 删除交易记录
- 平仓操作（自动计算盈亏）
- 支持多品种：工碳、电碳、矿石等
- 支持自定义品种

### 2. 数据筛选
- 按状态筛选（持仓中/已平仓）
- 按品种筛选
- 按时间范围筛选
- 按关联PO筛选

### 3. 统计分析
- 开仓价 vs SMM价格对比
- 结算价 vs SMM价格对比
- 折扣分析
- 盈亏统计
- 品种表现

### 4. 数据导入导出
- 导出为CSV文件
- 导入历史交易数据

## 数据存储
所有数据存储在 `data\trading_journal.db` SQLite数据库文件中

## 备份数据
定期备份 `data\trading_journal.db` 文件即可

## 常见问题

### Q1: 双击启动脚本后闪退？
A: 请确保已安装Python并添加到PATH环境变量

### Q2: 依赖安装失败？
A: 检查网络连接，或手动执行：`pip install -r requirements.txt`

### Q3: 如何更换端口？
A: 编辑 `app.py` 文件最后一行，将 `port=5000` 改为其他端口

### Q4: 忘记密码怎么办？
A: 本系统没有密码保护，直接访问即可
